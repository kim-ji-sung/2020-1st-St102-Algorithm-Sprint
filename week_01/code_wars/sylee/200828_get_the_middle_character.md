#### Get the Middle Character

## 문제설명

> 단어가 하나 주어지고, 단어의 가운데 문자를 출력한다. (단, 단어의 길이가 짝수이면 가운데 2개를 출력한다.)

## 나의 풀이 방법

 ```python
def get_middle(s):
    return s[len(s) // 2 - 1 : len(s) // 2 + 1] if len(s) % 2 == 0 else s[len(s) // 2]
 ```

> 문자열의 길이를 반환해주는 `len()`을 사용해서 문제를 풀었다.




## 다른 사람의 풀이 방법

```python
# Best Practices 1
def get_m(s):
    return s[(len(s) - 1) // 2 : len(s) // 2 + 1]
```

> 나와 유사하지만 수학적(?) 원리를 이용해 더 간단히 작성했다. 



```python
# Clever
def get_middle(s):
    while len(s) > 2:
        s = s[1:-1]
    return s
```

 > 길이가 2보다 클때,  문자열s의 양끝을 제외한다. 홀수의 경우 n, n-2, ... , 5, 3, 1 로 가운데 1글자가 반환되고
 >
 > 짝수의 경우 n, n-2, ... , 4, 2 로 가운데 2글자가 반환된다. 

> 참신한 방법이었다. 하지만 효율적이지는 않다고 생각이 든다.



## 배운 점

>  수학적 원리를 이용하여 코드를 조금 더 간단히 만들 수 있다.



## 반성할 점

> 특별히 없었다고 생각한다.
