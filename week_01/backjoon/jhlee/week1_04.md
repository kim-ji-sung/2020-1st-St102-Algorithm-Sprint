# 백준 4673 <셀프 넘버>

## 문제 설명
생성자가 없는 숫자를 셀프 넘버라 한다. 10000 이하의 셀프 넘버를 출력하는 문제


## 나의 풀이법
```c++

bool check[10001] = { false, };

* bool 자료형을 통해 self number라면 check
for (int i = 1; i <= 10000; i++) {
	int sum = i;
	int x = i;

	while (x != 0) {
		sum += (x % 10);
		x /= 10;
	}
	if (sum > 10000)
		continue;

	check[sum] = true;
}


* 셀프 넘버 출력 부분
for (int i = 1; i <= 10000; i++) {
	if (!check[i])
		cout << i << '\n';
}



## 반성할 점
처음에 결과값은 잘 나왔지만 마지막에 디버깅 에러가 났다.
check 변수의 배열의 길이보다 높은 값에 접근을 한 것이 원인으로 
셀프넘버를 체킹하는 과정에서 if (sum > 10000) continue; 조건을 안해주어 이를 추가 해주었다.


## 배운점
배열에 접근할 때 할당한 크기보다 큰 값에 접근하는지 항상 주의가 필요하다라고 다시 한 번 생각했다.
